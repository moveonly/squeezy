Prefer `decl_search`, `reference_search`, or `symbol_context` for bare-name symbol queries in Rust, Python, Java, C#, Go, C/C++, JavaScript/TypeScript, Ruby, PHP, Kotlin, Swift, Scala, and Dart files — they follow imports and re-exports that regex misses. Use `glob` for literal text or files outside Rust, Python, Java, C#, Go, C/C++, JavaScript/TypeScript, Ruby, PHP, Kotlin, Swift, Scala, and Dart. List workspace file paths matching a glob without reading file contents. Respects .gitignore by default; set include_ignored=true only when ignored paths are intentionally needed.
