Prefer `decl_search`, `reference_search`, or `symbol_context` for bare-name symbol queries in indexed source files — they follow imports and re-exports that regex misses. Use `grep` for literal text or for file types the graph does not index. Search text files under a workspace path. Respects .gitignore by default; set include_ignored=true only when ignored files are intentionally needed. Use output_mode=count or files_with_matches for broad exploration before reading content.
