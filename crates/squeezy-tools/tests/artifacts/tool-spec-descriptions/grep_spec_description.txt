Prefer `decl_search`, `reference_search`, or `symbol_context` for bare-name symbol queries in Rust, Python, Java, C#, Go, C/C++, JavaScript/TypeScript, Ruby, PHP, Kotlin, Swift, Scala, and Dart files — they follow imports and re-exports that regex misses. Use `grep` for literal text or files outside Rust, Python, Java, C#, Go, C/C++, JavaScript/TypeScript, Ruby, PHP, Kotlin, Swift, Scala, and Dart. Search text files under a workspace path. Respects .gitignore by default; set include_ignored=true only when ignored files are intentionally needed. Use output_mode=count or files_with_matches for broad exploration before reading content.
