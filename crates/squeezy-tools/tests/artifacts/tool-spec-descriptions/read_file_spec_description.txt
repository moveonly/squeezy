Prefer `decl_search`, `reference_search`, or `symbol_context` first for symbol-shaped queries in Rust, Python, Java, C#, Go, C/C++, and JavaScript/TypeScript files. Use `read_file` for free-form text, unsupported languages, or after the graph returned zero packets. Read a bounded byte slice from one workspace file and return its sha256 receipt. Use `read_file` once the graph (or a free-form `grep`) has produced a path and span.
