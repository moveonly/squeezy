Prefer `decl_search`, `reference_search`, or `symbol_context` for bare-name symbol queries in Rust, Python, Java, C#, Go, C/C++, JavaScript/TypeScript, Ruby, PHP, Kotlin, Swift, Scala, and Dart files — they follow imports and re-exports that regex misses. Use `read_file` for literal text or files outside Rust, Python, Java, C#, Go, C/C++, JavaScript/TypeScript, Ruby, PHP, Kotlin, Swift, Scala, and Dart. Read a bounded byte slice from one workspace file and return its sha256 receipt. Use `read_file` once the graph (or a free-form `grep`) has produced a path and span.
